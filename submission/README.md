## What the fork changes

The fork adds a new `contentops` service to the seller agent. A buyer agent can ask for a small-business content kit, seller agents compete, and the winning seller delivers a machine-readable micro-asset.

## What the agent sells

For `contentops bakery selling cakes by WhatsApp`, the seller returns:

- WhatsApp replies for welcome, quote, follow-up, close, and post-sale.
- A short content calendar.
- Adaptation checklist.
- JSON formatted for reuse by another agent.

## Settlement path

The target flow is:

WANT -> BID -> AWARD -> DEPOSITED -> DELIVERED -> RELEASED

The starter kit's Solana devnet escrow remains the settlement spine. No private keys or mainnet funds are included in this repository.
